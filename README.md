# Projet Programmation système avancée  -   Files de messages

**Auteur du projet :** Leonard NAMOLARU

**Description :** 
Implémentation de files de messages pour une communication entre des processus tournant sur la même machine, en utilisant la mémoire partagée, et sans l’aide de files de messages existantes (POSIX ou System V) ou des sockets UNIX. L’accès parallèle à la file de messages est possible avec une protection appropriée.
